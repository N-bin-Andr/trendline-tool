# trendline-tool
Улучшенная модификация графического объекта "Трендовая линия" - расширение для  TigerTrade платформы для Windows. 

## Основные свойства и возможности
- Отрезок, ограниченный двумя конечными ценовыми точками
- Возможность растягивать и сжимать линию тренда за конечные точки
- Горизонтальный вид при равенстве цен начальной и конечной точек
- Автоматическое выравнивание линии по горизонтали при удержании клавиши Shift во время изменения ценовой точки
- Цвет линии настраивается из палитры в меню настроек
- Меню настроек раскрывается двойным щелчком по  линии тренда
- Добавление текста на  линию тренда с возможностью включения через checkbox
- Настройка цвета, размера, начертания (жирный, курсив, наклонный) и шрифта текста
- Расположение текста: выше, ниже или по середине линии тренда
- При расположении по середине линия под текстом не отображается
- Отображение ценовых уровней в области начальной и конечной точек с настройками цвета
- Возможность удлинения концов линии тренда (расширить вправо, расширить влево)
- Интеграция с системой alerts TigerTrade для оповещений при пересечении ценовых уровней
